Testando HTML

Esse projeto tem o objetivo de demonstrar as minhas habilidades básicas com HTML e CSS rústicos

HTML

HTML "HyperText Markup Language", que significa Linguagem de Marcação de Hipertexto é como diz o nome uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

CSS "Cascading Style Sheet", que significa Folha de Estilo em Cascatas é uma maneira de estilizar, dando cores, formas e posicionamentos para todos os seus elementos do HTML.
